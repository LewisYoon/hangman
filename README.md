# Hangman Game

A simple web based hangman game built with JavaScript, HTML, and SCSS.

Guess the hidden word before game over

---

## How to play

Downlad and run HTML on live server or visit following link: https://lewisyoon.github.io/hangman

## Features

- Random word selection from JSON file

- Interactive on-screen keyboard

- Letter guessing system

- Hangman image updates on wrong guesses

- Score tracking (wins / losses)

- Used words history

- Restart game functionality

---

## Tech Stack

- HTML5

- SCSS

- JavaScript

- DOM Manipulation
